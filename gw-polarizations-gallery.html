Gravitational-Wave Polarization Modes

This page collects six animated visualizations of the polarization states allowed in a generic metric theory of gravity: the tensor modes (+ and ×), the vector modes (x and y), and the scalar modes (breathing and longitudinal).

Replace each placeholder path with your own GIF file in the gifs/ folder.

Plus (+)

Tensor polarization mode producing alternating stretching and squeezing along orthogonal axes.

Cross (×)

Tensor polarization mode rotated by 45 degrees with respect to the plus mode.

Vector x

Vector polarization mode producing shear-like motion along one transverse direction.

Vector y

Vector polarization mode producing shear-like motion along the orthogonal transverse direction.

Breathing

Scalar polarization mode with isotropic expansion and contraction of the ring of test particles.

Longitudinal

Scalar polarization mode involving compression and stretching along the propagation direction.
